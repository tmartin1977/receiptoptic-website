Packed with Smart Utilities

ReceiptOptic runs locally on your PC and mobile devices, bringing lightning-fast analytics to your personal and business finance ledger.

🔒

100% Offline Accuracy

Zero-config instant parsing via local overrides. Vendor signature matching secures 100% accuracy, working completely without a network connection.

⚖️

IRS Schedule C & Tax Savings

Automatically map itemized business expenses to Schedule C tax lines. Track YTD tax deductions and view estimated self-employment tax savings in real-time.

🤖

Gemini AI OCR Scanner

Connect your Gemini API key to parse complex or crumpled receipts. Instantly detects line items, tax amounts, and payment methods with high-precision AI.

Warranty & Return Tracker

Keep track of active warranties and return periods. Receive notifications before warranties expire and never miss a return deadline again.

Secure Cloud Backup

Sync your local SQLite database securely via Supabase. Keeps your records backed up in the cloud and synced across your PC and mobile devices.

Biometrics & Auto-Lock

Unlock the PC app securely using native Windows Hello (PIN, Face, or Fingerprint). Protects your finance ledger with automatic locking on background idle.

🛠️ Installation & Setup Guide

💻 Windows Desktop Setup:

  1. Download ReceiptOpticDesktop.zip using the button above.
  2. Extract the contents of the ZIP folder to any directory (e.g., C:\ReceiptOptic).
  3. Double-click ReceiptOptic.exe to launch the app instantly (no installation process required).
  4. Optional: Right-click ReceiptOptic.exe and select Send to -> Desktop (create shortcut) for easy access.

How to Install ReceiptOptic

Follow these simple steps to set up the desktop client on your Windows PC:

1

Download the ZIP

Wait for the download to complete saving `ReceiptOpticDesktop.zip` to your computer.

2

Extract the Files

Right-click the ZIP file and select "Extract All...". Do not run it from inside the compressed folder.

3

Launch the App

Open the extracted folder and double-click "ReceiptOptic.exe" to start.

4

Security Notice

If Windows SmartScreen shows a warning, click "More info" and then select "Run anyway" to run the app.

📲

Install ReceiptOptic

Add this app to your home screen for quick offline access.
